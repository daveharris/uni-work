Is your computer very slow or taking a long time to boot up?
A few fast and cheap changes can bring it back to up today's technology and reduce waiting ties hugely

These upgrades include:
  1. RAM
  2. Hard Disk
  3. Optical Drive
  4. Monitor

1. RAM

The largest speed increase can be done by increasing the memory (or RAM) size. This is the place where programs are loaded, so with more RAM, more programs can be run at once, meaning faster loading times. The boot-up time of the operating system can also be dramatically reduced. Memory usually comes in the following sizes:
  • 128MB
  • 256MB
  • 512MB
  • 1024MB/1GB
To run Microsoft Windows XP, we recommend at least 512MB of RAM. RAM can easily be upgraded and is relatively inexpensive.

2. Hard Disk

Hard drives are an integral part of the computer. This saves all your data and holds the data of the operating system. Over the last few years hard drives have become faster, which means that loading and boot times decrease. The standard speed of the hard drive used to 5400RPM (Revs Per Min), but today 7200RPM is the norm. Hard drives usually come in the following sizes:
  • 40GB
  • 60GB
  • 80GB
  • 120G
We recommend at least a 40GB drive, as Microsoft Windows will take up about 8-10GB over time, so that leave space for photos, movies etc. Hard drive installation is straigt-forward.

3. Optical Drive

Optical drives are CD and DVD drives. CD reader/writers usually came with new computers but today, DVD reader/writers are standard. They allow you to read data to install software, look at a friend's photos etc. They can also be used to write data to disc, so that you have a backup of it if something happens to your hard drive. The capacity of a CD is 700MB, while 4.7GB can fix on a DVD, so a DVD is a much better backup device. The discs come in writable (writable only once, called a CD-R or DVD-R) or re-writable (able to be erased and writen again, called CD-RW or DVD-RW).
Most current drives are able to read and write CD and DVD drives multiple times (if the correct discs are used).

4. Monitor

Computer Monitors show the output of the computer and enable you to interact with it. In modern computers, the monitor is connected to a video card, which is controlled by the computer itself. In recent years, LCD (or flat-panel) monitors have become cheaper and more popular. They give a clearer picture, give off less radiation and take up less space than a traditional CRT montitor.
The resolution is the number of pixels on the screen. The higher it is, the smaller the windows are and hence you have a larger working area. This also means that movies and pictures are of higher quality.
The usual sizes and resolutions are shown below:
  • 15", 800x600
  • 17", 1024x768 or 1280x1024
  • 19", 1280x1024
