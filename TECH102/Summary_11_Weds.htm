Seeing the Invisible

Images of anything

One can imagine making an image to display any data that varies from point to point in space.

For example, one might measure the temperature at points and displaying the data as a gray scale image with 0 C = black and 100 C = white.

Or measure the water content of flesh and displaying that as a gray scale image with black = driest and white = wet.

Or measure magnetic field as a function of position and display gray scale with large fields as white, small as black.

Or determine the density of electrons is a material and display as a gray scale image with white = high density, black low.

Or Measure the population density of a certain insect in a field and display as gray scale image with high density white, low black.

Things like these examples are done reqularly.

 

Gray scale image for temperature

Computer monitors will only display 256 levels of gray and there's no point in displaying more than that as we learned when we studied the human eye. Higher Resolution

Suppose I have more accurate thermal information. Suppose I have 12 bit data (4096 possible values). What then? My temperature resolution is now 100/4096 = 0.0244 C. For data analysis I will keep all that resolution, but for display on a computer screen I will compress to 8 bit data as follows:

12 bit temp value 8 bit value

0 – 15: 0; 15-31: 1; 32-47 2; … 4080-4095: 255

False Colours: A Colour Palette

We can make our displays clearer by assigning RGB values to the 256 different levels of gray I am using for display? For example, suppose I took that image and displayed it with red for 256, blue for zero, and shades of gray between? Called a colour palette. Name this one Red Hot

How do they do that?

An image is just an array of numbers that are interpreted as colours.

These images are 240x320 = 76800 array elements or pixels.

I could have used 76800 little thermometers to measure the temperature and then put the whole mess in an excel file. Then I could make an image from that data. But there's a better way as we shall see shortly.
